# Contribuindo

Obrigado por querer contribuir! Esta lista depende da comunidade para crescer e se manter atualizada.

## Como adicionar um recurso

1. Faça um **fork** deste repositório
2. Adicione o recurso na **categoria correta** do `README.md`
3. Siga o **formato** descrito abaixo
4. Abra um **Pull Request** com uma descrição curta do que está adicionando

## Formato

Cada entrada deve seguir o padrão:

```markdown
- `[Tipo]` [Nome do Recurso](URL) — Descrição curta. Autor/Instituição, ano.
```

### Badges de tipo

| Badge | Quando usar |
|---|---|
| `[PDF]` | Apostilas, documentos técnicos, livros digitais |
| `[Vídeo]` | Videoaulas, palestras, playlists |
| `[Curso]` | Cursos estruturados (com módulos/certificado) |
| `[Dataset]` | Dados abertos, séries históricas |
| `[Ferramenta]` | Software, calculadoras, aplicativos |
| `[Embrapa]` | Publicações oficiais da Embrapa (usar junto com outro badge) |
| `[Portal]` | Sites de referência com múltiplos recursos |

## Critérios de aceitação

- **Gratuito**: o recurso deve ser acessível sem custo
- **Qualidade**: de instituição reconhecida (universidade, Embrapa, órgão governamental, associação técnica) ou de autor com credenciais na área
- **Atualidade**: preferencialmente de 2010 em diante. Exceções para clássicos sem substituto, desde que justificado
- **Acessível**: o link deve funcionar e o conteúdo estar disponível publicamente
- **Em português**: preferencialmente em PT-BR. Recursos em inglês são aceitos se não houver equivalente em português, marcados com `[EN]`
- **Educacional**: foco em conteúdo de estudo e referência técnica, não em marketing de produtos

## O que NÃO incluir

- Conteúdo pago ou com paywall
- Material promocional de empresas
- Links para torrents ou conteúdo com direitos autorais violados
- Recursos desatualizados sem relevância histórica
- Canais ou páginas de influenciadores sem lastro técnico/acadêmico

## Categorias

Se seu recurso não se encaixa em nenhuma categoria existente, sugira a criação de uma nova na descrição do PR. A estrutura segue as grades curriculares de Eng. Agronômica e Zootecnia das principais universidades brasileiras.

## Link quebrado?

Se encontrou um link quebrado, abra uma **issue** informando:
- Qual link está quebrado
- Se encontrou a URL atualizada

O CI de verificação de links roda semanalmente e abre issues automaticamente para links mortos.

## Dúvidas?

Abra uma [issue](../../issues) com sua pergunta.
